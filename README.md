# ESENCIA

ESENCIA es una aplicación movil desarrollada con flutter que consume una API de naruto para mostrar información relacionada con los personajes

## Estructura del proyecto

    pages/

    themebuilder/
        -theme.dart
        -util.dart

    services/
        -api_services.dart

    entity/
        -actividad.dart


assets/

    icons/

    images/

    screenshots/
